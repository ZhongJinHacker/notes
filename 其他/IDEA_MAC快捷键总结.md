### IDEA MAC 快捷键



#### 1. Controll + Option + 'o' 删除import了 但未用到的包或类



#### 2.  函数体内 fori + Tab 生成

```java
		for (int i = 0; i < ; i++) {
			
		}
```



#### 3. 函数体内 sout + Tab 生成

```java
System.out.println();
```



#### 4. 对选中代码上下左右移动

```properties
代码右移: Tab

代码左移: Shift + Tab

代码上移: Shift + Alt + 方向键 上

代码下移: Shift + Alt + 方向键 下
```

#### 5. 大小写切换

```
Shift + Command + U
```
