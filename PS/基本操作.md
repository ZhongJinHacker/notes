#### 查看文件 放大 

```
ctrl + '+'
```

```
z + 鼠标左键向左移动
```





#### 查看文件 缩小

 ```
 ctrl + '-'
 ```

```
z + 鼠标左键向右移动
```



#### 切换抓手模式，拖动图片

```
space + 鼠标左键移动
```



#### 新建文档

```
ctrl + n
```



#### 撤回操作

```
ctrl + z
或
历史记录中选择 【如果没有历史记录窗口，可以在窗口中将其调出】
```



#### 前进（撤回撤回操作）

```
ctrl + shift + z
```
